# Trinetra Roadmap

## Milestone 1

### Scope

- CRUD foundation
- Dashboard
- Navigation

### Outcome

- Working product baseline for projects, tasks, and memories
- Feature-based structure established

## Milestone 2

### Scope

- Chat
- Agent Layer
- Memory Retrieval
- Vitest Tests

### Outcome

- Rule-based chat service with context loading and planner recommendations
- Test harness for agent services

## Milestone 3

### Scope

- OpenAI Integration
- Voice Foundation

### Outcome

- LLM-backed responses integrated behind service boundaries
- Initial voice input/output architecture in place

## Milestone 4

### Scope

- Tool Calling
- Browser Automation

### Outcome

- Agent can execute external actions through bounded tool interfaces
- Browser workflows automated for common tasks

## Milestone 5

### Scope

- Autonomous Personal Operator

### Outcome

- Trinetra can plan and execute multi-step tasks with minimal supervision
- System acts as a reliable personal operational layer
